Question-1: What are the different ways to select an element in the DOM?

In JavaScript, there are several ways to select elements from the DOM.

  1. getElementById()
  2. getElementByClassName()
  3. getElementByTagName()
  4. querySelector()
  5. querySelectorAll()

Question-2: What is the difference between innerHTML, innerText, and textContent ?

The innerHTML property retrieves or sets the HTML content inside an element, allowing HTML tags.
On the other hand, innerText only retrieves or sets the visible text inside an element.
Meanwhile, textContent retrieves all the text within an element, including hidden elements, without parsing HTML.

Question-3: What is event delegation in the DOM? in paragraph

Event delegation in the DOM is a technique where a parent element handles events for its child elements, even if they are added dynamically.Event delegation improves performance by reducing the number of event listeners in the DOM and is especially useful for handling dynamically added elements.

Question-4: What is event bubbling in the DOM?

Event bubbling in the DOM is a process where an event triggered on a child element propagates upward through its parent elements until it reaches the html element. Event bubbling is useful because it allows developers to use event delegation, where a parent element can handle events for its child elements.

Question-5: How do you create, add, and remove elements using JavaScript?

In JavaScript, elements can be dynamically created, added, and removed using DOM methods. To create an element, the document.createElement() method is used, which generates a new HTML element that can later be customized by setting properties like textContent or className.
